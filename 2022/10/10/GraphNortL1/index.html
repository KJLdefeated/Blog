Graph Models and Terminology

Terminology

Definition 1.1

  • A graph GGG consists of two sets: V(G)V(G)V(G), called the vertex set, and E(G)E(G)E(G), called the edge set. An edge, denoted xy, is an unordered pair of vertices. We will often use G=(V,E)G=(V,E)G=(V,E) as short-hand.

Definition 1.2

  • The number of vertices in a graph GGG is denoted ∣V(G)∣|V(G)|∣V(G)∣, or more simply ∣G∣|G|∣G∣. The number of edge is denoted ∣E(G)∣|E(G)|∣E(G)∣ or ∣∣G∣∣||G||∣∣G∣∣.

Definition 1.3

Let GGG be a graph

  • If xy is an edge, then x and y are the endpoints for that edge. We say x is incident to edge e if x is an endpoint of e.
  • If two vertices are incident to the same edge, we say the vertices are adjacent, denoted x yx~yx y. Similarly, if two edges share an endpoint, we say they are neighbors and the set of all neighbors of a vertex xxx is denoted N(x)N(x)N(x).
  • If two vertices (or edges) are not adjacent then we call them independent.
  • If a vertex is not incident to any edge, we call it an isolated vertx.
  • If both endpoints of an edge with the same vertex, then we say the edge is a loop.
  • If there is more than one edge with the same endpoints, we call these multi-edges.
  • If a graph has no multi-edge or loops, we call it simple.
  • The degree if a vertex vvv, denoted deg(v)deg(v)deg(v), is the number of edges incident to vvv, with a loop adding two to the degree. If the degree is even, the vertex is called even; if the degree is odd, then the vertex is odd.
  • If all vertices in a graph GGG have the same degree kkk, then GGG is called a k-regular graph.

Definition 1.4

  • A subgraph HHH of a graph GGG is a graph where HHH contains some of the edges and vertices of GGG; that is, V(H)⊆V(G)V(H) \subseteq V(G)V(H)⊆V(G) and E(H)⊆E(G)E(H) \subseteq E(G)E(H)⊆E(G).

Definition 1.5

  • Given a graph G=(V,E)G=(V,E)G=(V,E), an induced subgraph is a subgraph G[V′]G[V']G[V′] where V′⊆VV' \subseteq VV′⊆V and every abailable edge from GGG between the vertices in V′V'V′ is included.
  • We say HHH is a spanning subgraph if it contains all the vertices but not necessarily all the edges of GGG; that is, V(H)=V(G)V(H) = V(G)V(H)=V(G) and E(H)⊆E(G)E(H) \subseteq E(G)E(H)⊆E(G).

Digraphs

Definition 1.6

  • A directed graph, or digraph, is a graph G=(V,A)G=(V,A)G=(V,A) that consists of a vertex set V(G)V(G)V(G) and an arc set A(G)A(G)A(G). An arc is an ordered pair of vertices.

Definition 1.7

Let G=(V,A)G = (V,A)G=(V,A) be a digraph

  • Given an arc xy, the head is the starting vertex x and the tail is the ending vertex y.
  • Given a vertex x, the in-degree of x is the number of arcs for which x is a tail, denoted deg−(x)deg^{-}(x)deg−(x). The out-degree pof x is the numver of arcs for which x is the head, denoted deg+(x)deg^{+}(x)deg+(x).
  • The underlying graph for a digraph is the graph G′=(V,E)G'=(V,E)G′=(V,E) which is formed by removing the direction the direction from each arc to form an edge.

Weighted Graphs

Definition 1.8

  • A weighted graph G=(V,E,w)G=(V,E,w)G=(V,E,w) is a graph where each of the edges has a real number associated with it. This number is referred to as the weight and denoted w(xy)w(xy)w(xy) for the edge xyxyxy.
